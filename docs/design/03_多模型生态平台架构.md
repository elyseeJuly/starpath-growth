# 多模型生态平台架构原型 v1.0

## 一、架构概述

本架构旨在建立一个**模拟+真实混合**的ASD社会化辅助支持系统生态，包含：

1. **多智能体协作引擎** - 核心能力画像与干预生成
2. **场景模拟系统** - 用于演示和训练
3. **真实场景接入层** - 咖啡馆等实际应用入口
4. **开放接口层** - 预留扩展能力

### 4.1 模拟场景类型
| 场景 | 模拟内容 | 验证指标 |
|------|---------|---------|
| **社交互动** | 模拟顾客点单、日常对话 | 指令理解度、回应准确性 |
| **任务执行** | 咖啡机操作、清洁流程 | 步骤完成率、错误次数 |
| **情绪调节** | 模拟压力场景 | 情绪稳定性、调节策略 |
| **环境适应** | 场景切换、人员变化 | 适应时间、错误率 |

## 五、开放接口设计

### 5.1 REST API端点
| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/profile/{id}` | GET | 获取患者能力画像 |
| `/api/profile/{id}` | PUT | 更新患者画像 |
| `/api/records` | POST | 录入行为记录 |
| `/api/intervention` | POST | 请求干预建议 |
| `/api/simulation` | POST | 运行模拟场景 |

### 5.2 数据导入接口
- 支持CSV/Excel批量导入
- 支持JSON格式API对接
- 支持增量同步

### 5.3 三方系统对接
- 康复机构管理系统
- 可穿戴设备数据
- 智能家电接口（预留）

## 六、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | React + TypeScript | 组件化开发 |
| 可视化 | D3.js / ECharts | 画像展示、图表 |
| 后端 | Node.js + Express | API服务 |
| 数据库 | PostgreSQL + Redis | 数据存储 |
| 智能体框架 | LangChain / LangGraph | 多智能体协作 |
| 部署 | Docker + Docker Compose | 容器化运行 |

## 七、模拟演示流程

```
1. 选择模拟场景（如：咖啡馆点单）
2. 设置患者初始能力等级
3. 运行模拟交互
4. 智能体实时评估并给出干预建议
5. 查看模拟报告和成长曲线
6. 对比不同干预策略效果
```
