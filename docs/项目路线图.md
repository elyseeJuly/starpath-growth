# StarPath - ASD社会化辅助支持系统 项目路线图

## 项目愿景

建立一套像"盲道"和"AED"一样的基础设施，帮助ASD患者更好地融入社会。

---

## 总体路线

```
Phase 0: 画像基础 → Phase 1: 咖啡馆MVP → Phase 2: 核心框架 → Phase 3: 生态系统
```

---

## Phase 0: 画像基础（当前阶段）

**目标**：建立能力画像体系

**完成内容**：
- ✓ 能力维度模型（8大维度，40+子维度）
- ✓ 咖啡馆MVP数据采集规范
- ✓ 技术架构选型

**下一步**：
1. 数据模型Schema定义
2. 项目工程初始化
3. 纸质记录试点（可选）

---

## Phase 1: 公益咖啡馆MVP

**目标**：在真实场景验证画像系统

**时间**：1-3个月

**功能**：
1. 康复老师数据录入界面
2. 患者基础信息管理
3. 简单能力画像展示
4. 成长曲线可视化

**交付物**：
- Web版数据录入系统
- 第一批真实数据
- 初步验证报告

---

## Phase 2: 核心框架v1.0

**目标**：建立基于画像的干预系统

**时间**：3-6个月

**功能**：
1. 多源数据融合引擎
2. 简单画像推理算法
3. 个性化干预建议生成
4. 多方协作界面

**交付物**：
- 核心框架系统
- 画像引擎API
- 干预方案模块

---

## Phase 3: 生态系统扩展

**目标**：覆盖更多场景，形成支持网络

**时间**：6-12个月

**扩展**：
1. 更多职业场景（超市、图书馆等）
2. 社区生活支持
3. 家庭训练工具
