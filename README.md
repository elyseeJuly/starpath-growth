# StarPath - ASD社会化辅助支持系统

> 面向成年ASD患者的社会交互理解对齐多智能体框架

## 🎯 项目目标

为ASD（自闭症谱系障碍）患者提供一个社会化辅助支持系统，帮助他们提升社交能力、职业技能和生活质量。

## ✨ 核心功能

### 1. 能力画像系统
- 8大能力维度评估（社交、认知、生活自理、情绪管理、沟通、职业、适应、感知觉）
- 动态更新的个性化能力档案
- 智能干预建议生成

### 2. 多方协作平台
- **患者端**：任务执行、情绪打卡、场景模拟
- **家长端**：进度查看、辅助训练
- **康复老师端**：数据录入、干预管理
- **管理端**：用户管理、数据分析

### 3. 场景模拟引擎
- 咖啡馆岗位适应模拟
- 社交互动场景训练
- 任务执行能力评估

### 4. 个性化引导系统
- 根据角色提供不同级别的引导
- 患者：全面引导（语音提示、视觉高亮、步骤动画）
- 家长/老师：简化引导

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Node.js 16+
- npm 8+

### 启动方式

**方式一：使用启动器（推荐）**

```bash
# macOS/Linux
./start.sh

# Windows
start.cmd
```

**方式二：手动启动**

```bash
# 启动后端
cd backend
source venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# 启动前端（新开终端）
cd frontend
npm install
npm start
```

### 访问地址
- 🌐 前端: http://localhost:3000
- 🔌 API: http://localhost:8000
- 📖 API文档: http://localhost:8000/docs

## 👥 角色说明

| 角色 | 权限 | 功能 |
|------|------|------|
| 患者 | 最低 | 任务执行、情绪打卡、场景模拟 |
| 家长 | 中等 | 查看进度、辅助训练 |
| 康复老师 | 较高 | 数据录入、干预管理 |
| 管理员 | 最高 | 用户管理、系统配置 |

## 📊 能力维度

| 维度 | 描述 |
|------|------|
| 社交能力 | 与人交往、理解社交规则 |
| 认知能力 | 注意力、记忆力、问题解决 |
| 生活自理 | 饮食起居、自我照顾 |
| 情绪管理 | 情绪识别与调节 |
| 沟通能力 | 语言表达与理解 |
| 职业能力 | 完成工作任务 |
| 适应能力 | 适应新环境 |
| 感知觉 | 感官处理与整合 |
