# StarPath 启动器使用说明

## 🚀 使用方法

### 方式一：直接双击启动器（推荐）

1. 在 Finder 中找到 **"StarPath Starter.app"**
2. 双击即可启动系统

### 方式二：使用终端脚本

1. 打开终端
2. 进入项目目录：
   ```bash
   cd /path/to/starpath\ growth
   ```
3. 运行启动脚本：
   ```bash
   ./start_starpath.sh
   ```

---

## 📍 文件夹结构

```
starpath growth/
├── StarPath Starter.app/          ← 双击这个启动 (macOS应用)

```

## 🛑 停止服务

启动后终端窗口会保持打开，按 **Enter** 键即可停止所有服务。

---

## ⚠️ 常见问题

### 1. macOS 提示"无法打开"
如果是首次下载，需要在终端运行：
```bash
xattr -d com.apple.quarantine "/path/to/StarPath Starter.app"
```

### 2. 端口被占用
```bash
# 查看端口占用
lsof -i :3000
lsof -i :8000

# 结束进程
kill -9 <PID>
```

### 3. 需要重新安装依赖
删除以下文件夹后重新启动：
- `backend/venv/`
- `frontend/node_modules/`

---

## 💡 提示

- 可以将 **StarPath Starter.app** 拖到 Dock 栏方便快速启动
- 首次启动需要等待几分钟安装依赖
- 保持终端窗口不要关闭
